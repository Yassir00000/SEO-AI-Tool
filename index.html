KeywordPlannerAI

AI-powered tool integrating Google Keyword Planner API to generate SEO-oriented content

        $ keywordplannerai generate --topic="travel tips"

Installation

  • npm
  • apt
  • choco
            $ npm install -g keywordplannerai
            $ sudo apt-get install keywordplannerai
            $ choco install keywordplannerai

Real-time Keyword Data

Fetch up-to-date search volume, competition, and trend data directly from Google Keyword Planner.

SEO Content Generation

Auto-generate optimized blog posts, meta descriptions, and titles using your keywords.

Customizable Templates

Choose from pre-built content templates or define your own structures.

Easy Configuration

Configure API keys and default settings in a single config.json file.

Analytics Dashboard

View keyword performance reports and content effectiveness at a glance.

Seamless Integration

Integrate with CMS platforms, export to CSV, or pipe into your own workflows.

    API Endpoints

  • GET /keywords?topic={topic}
  • POST /generate-content
  • GET /reports/{reportId}

    CLI Commands

  • keywordplannerai analyze Analyze keyword metrics
  • keywordplannerai generate Generate SEO content
  • keywordplannerai report View past reports
